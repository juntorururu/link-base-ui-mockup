# LINK MALL Search UI Mockup

LINK MALL TOPページの検索機能強化を開発担当へ共有するための静的UIモックアップです。

## 公開URL

https://juntorururu.github.io/link-base-ui-mockup/mall-search/

## 収録した検索機能

- キーワード検索
- サジェスト（オートコンプリート）
- 検索履歴
- カテゴリ検索
- ショップ・事業者検索
- 価格帯指定
- 送料無料、まとめ出荷対応、送料無料ライン対象のトグル
- レビュー評価
- 認定掲載者条件
- 並び替え条件

## デザイン方針

- LINK MALLの白基調に合わせた青緑系アクセント
- 購入・検索導線に馴染むオレンジを補助色として使用
- 添付参考画像のように、上部検索、カテゴリ切替、条件指定をスマホ画面内で見せる構成
- 既存のルート `index.html` は残し、検索UIモックは `mall-search/` 配下に追加
